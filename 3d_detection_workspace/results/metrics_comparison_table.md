## Key Takeaways

1. **mAP/AP**: Measures overall detection accuracy
2. **Precision**: Accuracy of positive predictions
3. **Recall**: Coverage of ground truth objects
4. **IoU**: Average overlap with ground truth boxes
5. **FPS**: Inference speed (frames per second)
6. **Latency**: Time per frame in milliseconds
7. **GPU Memory**: Peak memory usage during inference
